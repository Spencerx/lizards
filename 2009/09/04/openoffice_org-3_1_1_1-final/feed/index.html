]]> By: Petr Mladek https://lizards.opensuse.org/2009/09/04/openoffice_org-3_1_1_1-final/#comment-1233 Mon, 07 Sep 2009 10:38:59 +0000 http://lizards.opensuse.org/?p=2067#comment-1233 The version 3.1.1 provides KDE4 integration. Unfortunately, it is available only for openSUSE-Factory because it required a newer Qt. It is a bug that it does not fall back to KDE3 integration when the KDE4 plugin is not available. Could you please report it into bugzilla?
You might force the KDE3 plugin by the environment variable OOO_FORCE_DESKTOP. You might call OOo by:

OOO_FORCE_DESKTOP=kde3 ooffice

Also you might set it in your system by default. It should be enough to add the following line into $HOME/.bashrc and relogin:

export OOO_FORCE_DESKTOP=kde3

Regarding the signature, it is automatically generated by the Build Service framework. I expect that the older signature was out of date. I am going to ask the BS team to confirm this.
Thanks a lot for feedback.
