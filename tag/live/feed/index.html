Live – openSUSE Lizards https://lizards.opensuse.org Blogs and Ramblings of the openSUSE Members Fri, 06 Mar 2020 11:29:40 +0000 en-US hourly 1 openSUSE-Education Live Media including official updates https://lizards.opensuse.org/2009/04/24/opensuse-education-live-media-including-official-updates/ https://lizards.opensuse.org/2009/04/24/opensuse-education-live-media-including-official-updates/#comments Fri, 24 Apr 2009 13:25:56 +0000 http://lizards.opensuse.org/?p=743 Currently, the openSUSE-Education team is “playing” with the new features of the external build service. As a first result, we’re proud to announce our new live/installable DVD!

This DVD includes:

  • the official updates to all packages since the openSUSE 11.1 release
  • fully pre-configured, ready to run KIWI-LTSP server
  • tons of applications from openSUSE Education repository

You can get it from here: http://download.opensuse.org/repositories/Education/images/iso/ – it’s called openSUSE-Edu-KIWI-LTSP-live*.iso (the openSUSE-Edu-KIWI-LTSP-live-unstable*.iso is for development only). Please watch for the Build Numbers at the end of the ISO-Name if you report bugs in our bugzilla.

Have a look at CyberOrgs blog on the most efficient ways to download the image.
