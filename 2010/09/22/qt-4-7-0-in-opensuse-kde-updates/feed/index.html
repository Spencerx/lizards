KDF = KDE:Distro:Factory (KDE 4.5 snapshots, built vs Qt 4.7 as of today)

KDR is no repo I maintain, I guess it’s a typo, i guess he meant to write KDF.

KDS = KDE:Distro:Stable (11.3 packages of KDE 4.4.)

KR45 = KDE:Release:45 (KDE 4.5 releases, built vs Qt 4.6)

KUSC = KDE:Unstable:SC (What will become KDE 4.6)

KUP = KDE:Unstable:Playground (Random unstable package snapshots and betas)
